Home of the Geeks

CS3, the Intro computer survey class of Santa Monica College, is offered every semester. Several
sections are available. Nevertheless, the popularity of the class has meant in the past that not all students
waiting to take the class could be admitted.

The primary textbook for the course is Computers, Tools for and Information Age by H.L Capron.

Click here for informational class web page

The class covers several areas of study:

  • The Central Processing Unit
  • Input and Output
  • Storage
  • Networking
  • Processing
  • Spreadsheets
  • Programming (Visual Basic)
